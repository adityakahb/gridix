AEM Data Attributes

Both tables below are fully configured through data-gridix-* attributes on the <table> element and data-gridix-col-* attributes on individual <th> elements. An AEM dialog author sets these values — no JavaScript required on the page. The CDN build (gridix.cdn.min.js) auto-initialises every [data-gridix] table on DOMContentLoaded.

Table 1 — Search & sort only, no pagination

Configured for a short reference table where pagination would be unnecessary.

<table
  data-gridix
  data-gridix-search="true"
  data-gridix-pagination="false"
  data-gridix-sort="true"
  data-gridix-export="csv,copy"
  data-gridix-empty-text="No countries match your search."
>

Table 3 — data-gridix-empty-text

The table below starts empty so the message is visible immediately. Try typing in Table 1's search box to produce zero results and see the same mechanism live.
