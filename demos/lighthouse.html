Optimisation techniques applied

  • CSS preloaded — <link rel="preload"> eliminates render-blocking.
  • Deferred JS — Gridix script uses type="module" (automatically deferred).
  • Semantic HTML — role="grid", aria-label, <caption>, scope="col" on headers.
  • No layout shift (CLS) — wrapper dimensions are stable; toolbar insertion uses reserved space.
  • Reduced motion — all transitions are wrapped in prefers-reduced-motion.
  • Touch targets — minimum 44×44 px on coarse-pointer devices (WCAG 2.5.5).
  • Colour contrast — all text/background pairs meet WCAG AA (≥ 4.5:1).
  • Focus management — roving tabindex, escape key, programmatic focus on page change.
  • sideEffects: false — ESM / CJS builds are fully tree-shakeable.
  • Sourcemaps — unminified builds ship with external sourcemaps for DevTools debugging.
