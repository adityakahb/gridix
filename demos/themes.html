Themes

Three built-in themes set via data-gridix-theme. All colours are CSS custom properties (--gridix-accent, --gridix-color-border, etc.) so any theme can be overridden per-component in AEM without any build step — demonstrated in the custom override section below.

Custom override via CSS custom properties

Setting --gridix-accent: #8b5cf6 and --gridix-color-header-bg: #faf5ff on a wrapper element re-themes the table without touching the theme attribute. In AEM this can be done via an inline style on the component container.
