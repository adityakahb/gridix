Responsive Widths

Each table below measures its own natural content-fit width using the browser's layout engine, then sets that as its min-width. The scroll container only shows a scrollbar when the viewport is narrower than that specific table's natural width — not a shared fixed percentage applied to all tables alike.

Resize the window (or use browser DevTools device emulation) to see each table respond independently.

How it works The plugin temporarily sets each table to width:auto; table-layout:auto, reads scrollWidth (the browser's computed minimum), then restores styles and sets that value as the table's min-width. A ResizeObserver re-runs this on container resize.
